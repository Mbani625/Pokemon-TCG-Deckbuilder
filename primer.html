Markdown Help
  • Headers: Use # for headers (e.g., # Header 1, ## Header 2).
  • Bold: Use **text** for bold text.
  • Italic: Use *text* for italic text.
  • Lists: Use - item for unordered lists and 1. item for ordered lists.
  • Links: Use [title](url) for hyperlinks.
  • Images: Use ![alt text](url) for images.
  • Code: Use backticks `code` for inline code or triple backticks for code blocks.
